# Salesforce AppExchange Apps Scraper & Market Intelligence

Apify Actor for discovering and extracting Salesforce AppExchange app data, with structured outputs for market intelligence, competitive analysis, and reporting.

## Features

- App discovery by category and sphere
- Structured app extraction
- CSV / XLSX style export-ready data
- JSON market intelligence outputs
- Executive summary and PDF reporting support

## Tech Stack

- Python
- Playwright
- Apify
