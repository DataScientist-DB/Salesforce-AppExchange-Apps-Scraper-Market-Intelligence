This folder contains sample datasets generated by the Salesforce AppExchange Discovery Engine.

APPS_SAMPLE.csv
- Includes 10 sample apps with realistic metadata fields.
- Shows the exact structure of the output produced by the Actor.
- Contains fields such as app_id, name, vendor, rating, price model, listing URL.

REVIEWS_SAMPLE.csv
- Includes 20 sample reviews linked to the sample apps.
- Demonstrates review fields: rating, title, author, date, and review text.
- Matches the real output layout from the Actor’s reviews mode.

These files are for demonstration only and help users understand the output structure before running the Actor.
